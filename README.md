# AutoMark Story Generator Static

نسخة static جاهزة للرفع مباشرة على أي hosting بدون Python أو backend.

## الملفات المطلوبة

- `index.html`
- `styles.css`
- `app.js`
- `assets/`

## الرفع السريع

### Netlify Drop

1. افتح https://app.netlify.com/drop
2. اسحب فولدر `story-grid-maker-static`
3. الموقع يطلع live فورًا.

### GitHub Pages

1. اعمل repository جديد.
2. ارفع محتويات فولدر `story-grid-maker-static`.
3. من Settings > Pages اختار branch `main` و root.
4. أي تعديل على GitHub هيظهر على الموقع بعد deploy.

## ملاحظات

- التحليل هنا local داخل المتصفح فقط.
- مفيش Claude Vision API في النسخة static.
- زر التحميل يعتمد على browser download مباشرة.
